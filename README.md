# PHP Sample

## Introduction

An PHP MVC Web Application

##Prerequisite

- PHP 8.x
- MySQL 5.x
- NPM Pakage Manager
## Installation

- Download XAMPP.
- Clone this repo.
- Run `npm install`
- Run `npm run css:build` to build css
- Run `C:\xampp\php\php.exe -S localhost:3030 -t {{your_directory}}\{PHPSample}`

## Visit site
- `localhost:3030/index.php`
